Class: BallMaker

BallMaker(actualCol, actualColData)

Class creating balls for a particular year in the dataset.

Constructor

new BallMaker(actualCol, actualColData)

This class represents the circles that will be made for the svg. The reason I have used collision detection is to model the particles in the atmosphere that collide with each other in the air. This class contains year and gases of that year parameters - used to create svg from the dataset.
Parameters:
Name Type Description
actualCol string allows the year selected from the dropdown menu to be used in creating the svg from the data of that year.
actualColData string is the data of the specific year which has been put into an array.

(static) runforce(radiusSize, colour)

Function runforce is called to produce the balls in svg form. The visualisation code involving the gravitational force between the balls and collisions is included in this function as well as implementing the properties for each ball. The number of nodes (balls) available has been fixed to 31, to give a representation of the concentation of gases in air. This function removes the previous svg and adds the new one created for another year with the parameters passed into the function.
Parameters:
Name Type Description
radiusSize string holds the list of ball radii - carried in from runSVG()
colour string allows the colours in the list to be used to colour the corresponding ball with the radiusSize (size of radius)

runSVG()

Function that is called when an object of BallMaker has been made. Allows the amount of balls for each colour to be calculated along with the radii of each ball. Sets the coloured labels to the correct data retrived from the data set. This function also includes loops that check how many balls will be required in a particular colour, i.e how many balls should be coloured orange with a given amount of data in a particular year.
